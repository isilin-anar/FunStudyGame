# 🐍 PYTHON MATRIX

> 파이썬을 누구나 쉽고, 매일매일, 무료로 배울 수 있는 웹사이트

---

## 🎯 프로젝트 목적

돈이 없어 학업을 못하는 아이들도 인터넷만 있으면 파이썬을 배울 수 있도록
완전 무료로 운영되는 파이썬 학습 플랫폼입니다.

---

## 📚 주요 기능

- ✅ 완전 기초(변수, 자료형)부터 고급 심화까지 단계별 학습
- ✅ 이론 → 퀴즈 → 실습 순서의 체계적인 학습 구성
- ✅ 매일 타이핑 연습을 위한 **코드 낙하 게임** (3분 타이머)
- ✅ 난이도 선택 기능 (초초보자 ~ 어려움)
- ✅ 매트릭스 스타일의 몰입감 있는 UI

---

## 🚀 사용 방법

별도 설치 없이 브라우저에서 바로 실행 가능합니다.
```
index.html 파일을 브라우저로 열거나,
배포된 웹사이트 주소로 접속하세요.
```

---

## 🛠️ 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript (외부 라이브러리 없음)

---

## 📋 학습 단계 구성

| 단계 | 주제 | 상태 |
|------|------|------|
| 1단계 | 변수와 자료형 | ✅ 완료 |
| 2단계 | 리스트와 반복문 | 🔧 개발 중 |
| 3단계 | 조건문과 함수 | 🔧 개발 중 |
| 🎮 게임 | 코드 낙하 게임 | ✅ 완료 |
